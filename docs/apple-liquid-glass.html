🎨 核心 CSS 属性

💧 基础毛玻璃

使用 backdrop-filter: blur() 创建基础的毛玻璃效果。

backdrop-filter: blur(20px);

✨ 高级毛玻璃

增加 saturate() 提升色彩饱和度，配合多层阴影。

backdrop-filter: blur(40px) saturate(180%);

🎛️ 控制中心风格

更强的模糊和饱和度，模拟 iOS 控制中心效果。

backdrop-filter: blur(50px) saturate(200%);

🔔 通知卡片风格

更高的透明度和饱和度，适合通知和浮层。

backdrop-filter: blur(60px) saturate(200%);

📚 技术要点

🔑 核心属性

backdrop-filter 是实现毛玻璃效果的核心 CSS 属性。

  • blur() - 模糊程度
  • saturate() - 饱和度
  • brightness() - 亮度

⚠️ 浏览器兼容性

需要添加 -webkit- 前缀以支持 Safari。

  • Chrome 76+
  • Safari 9+ (需前缀)
  • Firefox 103+
  • Edge 79+

💡 最佳实践

  • 使用半透明背景色
  • 配合多层阴影增加深度
  • 使用 inset 阴影模拟高光
  • 添加边框增加层次感

🎯 应用场景

  • 导航栏和工具栏
  • 卡片和弹窗
  • 通知和提示
  • 控制面板和设置
