# Embera · 余温 — 首日冷启动体验剧本

> 目标：用户在10分钟内产生一次真实情绪波动（哪怕只是轻微心酸、被理解、被记住）

---

## 🪶 STEP 0：打开 App（前3秒）

**禁止**：登录注册、选角色、填信息

**正确做法**：
1. 黑屏
2. 慢慢亮起（3秒渐变）
3. 中间一行字：「有些人，不会真的离开。」
4. 停2秒
5. 淡入：「你想见谁？」
6. 进入人物选择页

---

## 🫧 STEP 3：第一次互动（关键分水岭）

用户点击屏幕 → 输入框出现：「想说点什么吗…」

### ❗ 关键机制：第一句必须触发"轻击中"

### 用户输入 → AI 回复对照表

#### 场景A：用户说「你好」

| 角色 | 回复 |
|------|------|
| 稳重哥哥 | 你好。不用紧张。 |
| 温柔学姐 | 你好。今天…怎么样？ |
| 元气少女 | 你好你好！嗯…有点正式呢。 |

**记忆钩子**：记录「用户开场比较谨慎」

---

#### 场景B：用户说「有点累」

| 角色 | 回复 |
|------|------|
| 稳重哥哥 | 嗯…听起来不是一点点累。 |
| 温柔学姐 | 今天是不是撑太久了。 |
| 元气少女 | 没关系！累的话…先停一下也可以。 |
| 傲娇大小姐 | 累还来找人说话，真是… |
| 知性御姐 | 累，是正常的。不用解释。 |
| 阳光学长 | 走，我们慢慢来。 |
| 腹黑总裁 | 你不是累，是在扛。 |
| 年下弟弟 | 那你可以先不说话，我陪你一下。 |

**记忆钩子**：记录「用户最近很累」→ 3轮后回调

## 🧠 STEP 4：第一次"被记住"（第3-5分钟）

**系统行为**：抓住用户一句话 → 存入 memory

**触发时机**：第3-5轮对话

**回调示例**：

> 用户之前说：「最近工作很烦」

| 角色 | 回调 |
|------|------|
| 稳重哥哥 | 刚刚你说工作很烦…现在好一点了吗？ |
| 温柔学姐 | 你之前说工作很烦…现在呢？ |
| 元气少女 | 诶，你刚才说工作很烦！现在好点没？ |

**用户心理**：「它真的在听我说话」
