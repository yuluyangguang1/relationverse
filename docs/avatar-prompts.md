## 使用方法

### Midjourney
1. 复制 prompt 粘贴到 Discord
2. 每个角色生成 4 张，选最好的
3. 用 `/imagine` 命令

### Stable Diffusion (本地/在线)
1. 推荐模型：**RealVisXL v4.0** 或 **Juggernaut XL**
2. 推荐设置：Steps: 30, CFG: 5-7, Sampler: DPM++ 2M Karras
3. 分辨率：1024×1024

### 即梦/可灵 (国内免费)
1. 把英文 prompt 翻译成中文直接用
2. 选择"写实人像"风格

---

## 生成后处理
所有图片统一处理：
1. 裁切为正方形 1:1
2. 人脸居中（眼睛在 1/3 线）
3. 缩放到 400×400px
4. 存放在 `frontend/src/assets/avatars/` 目录
