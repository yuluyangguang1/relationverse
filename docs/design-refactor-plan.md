# Embera · 余温 设计重构方案

## 设计趋势研究总结

### 从参考网站学到的

| 网站 | 核心启示 |
|------|---------|
| **curated.design** | 暗色模式 + 渐变光晕 + 大留白，情感类产品用暖色调比冷色更打动人 |
| **onepagelove.com** | 单页应用趋势：极简排版、高质量摄影头像、视差滚动、玻璃拟态进化版 |
| **component.gallery** | 组件设计：大圆角卡片、清晰层级、无衬线字体、柔和阴影 |
| **mobbin.com** | 移动端：底部导航悬浮感、卡片堆叠、大点击区域、圆角至上 |
| **rebrand.gallery** | 品牌趋势：极简符号Logo、高饱和度+深色背景、无衬线主导、复古衬线体点缀 |

### 2024-2025 关键趋势

1. **暗色模式为主流** — 但不是纯黑，而是带色偏的深蓝灰（#0a0a14 / #0d0d1a）
2. **液态玻璃 2.0** — 从早期毛玻璃进化为：微妙折射 + 内发光 + 边缘高光
3. **温暖渐变取代冷科技色** — 橙→粉→紫的暖色渐变，传递情感温度
4. **超大圆角** — 卡片 border-radius: 20-28px，按钮 12-16px
5. **大量留白 + 呼吸感** — 元素间距是过去的 1.5-2 倍
6. **真实摄影取代插画** — AI生成的高质量人像成为情感产品标配
7. **微动效** — 页面切换 0.3s ease-out、卡片 hover 上浮、按钮发光脉冲
8. **情感化空状态** — 不再冷冰冰的"暂无数据"，而是有温度、有插画的引导

---

## Embera 当前问题诊断

1. **配色偏冷** — 紫色主导，缺少"余温"的暖感
2. **卡片拥挤** — 圆角太小、间距不够、边框过重
3. **玻璃效果粗糙** — blur值过高，没有内发光和边缘折射
4. **排版层级弱** — 标题/正文/辅助文字区分度不够
5. **动效缺失** — 页面切换生硬，没有hover反馈
6. **按钮简陋** — 没有渐变边框、发光、按压效果
7. **缺少情感化细节** — 加载状态、空状态、过渡动画

## 重构清单

### Phase 1: 配色系统重塑
- [ ] 背景色：#0a0a14 → #06060f（更沉的深空黑）
- [ ] 卡片背景：rgba(255,255,255,0.03) → 带色偏的 rgba(255,245,230,0.04)
- [ ] 主色：紫 → 暖橙 #FF9E5E + 柔粉 #FF5E78
- [ ] 辅助色：增加琥珀金 #FFD9A8 作为高亮
- [ ] 文字层级：白 #fff → 灰白 #e5e5e5 → 暗灰 #888 → 更暗 #555

### Phase 2: 卡片 & 容器
- [ ] 圆角：12px → 24px
- [ ] 边框：1px solid rgba(255,255,255,0.06) → 更微妙的 rgba(255,158,94,0.08)
- [ ] 阴影：去掉生硬黑色阴影 → 添加暖色光晕阴影
- [ ] 间距：padding/margin 统一增大 50%

### Phase 3: 液态玻璃 2.0
- [ ] backdrop-filter: blur(20px) → blur(12px) + saturate(150%)
- [ ] 添加内阴影：inset 0 1px 1px rgba(255,255,255,0.1)
- [ ] 添加边缘折射：border-top 1px rgba(255,255,255,0.15)
- [ ] 背景渐变叠加：linear-gradient(180deg, rgba(255,245,230,0.05), transparent)

### Phase 4: 排版升级
- [ ] 标题：font-weight 700 → 600（更优雅）
- [ ] 增加 letter-spacing: -0.02em（紧凑现代感）
- [ ] 行高：1.5 → 1.6（更透气）
- [ ] 增加字体层级：Display / Title / Body / Caption / Overline

### Phase 5: 动效系统
- [ ] 页面切换：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- [ ] 卡片 hover：translateY(-4px) + 阴影扩散 + 边框发光
- [ ] 按钮 hover：渐变偏移 + 外发光 pulse
- [ ] 加载状态：火焰Logo脉动动画
- [ ] 列表入场：stagger 0.05s fade-in + translateY

### Phase 6: 组件精致化
- [ ] 按钮：渐变背景 + 渐变边框 + hover发光
- [ ] 输入框：聚焦时暖色光晕
- [ ] 头像：增加微妙发光环
- [ ] 底部导航：悬浮胶囊设计 + 活跃项发光
- [ ] 进度条：渐变填充 + 微光扫过动画

### Phase 7: 情感化细节
- [ ] 空状态："还没有人陪伴你，创建第一个角色吧" + 插画
- [ ] 加载中：火焰Logo呼吸动画 + "余温正在汇聚..."
- [ ] 聊天气泡：发送时有轻微弹跳
- [ ] 亲密度变化：数字跳动动画 + 进度条流光
